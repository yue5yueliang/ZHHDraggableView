## 简介

`ZHHDraggableView` 让你可以将任意视图变为可拖动并悬浮在屏幕上的元素，类似于 iOS 中的 AssistiveTouch。支持自定义拖动方向、边界控制等功能，可以广泛应用于悬浮按钮、拖动视图等场景。
